Instrucciones para ejecución de algoritmo forward checking + HOD

1) Ejecutar make, esto generará el archivo main.o
2) El algoritmo se ejecuta con el comando ./main "nombre_instancia", por ejemplo si la instancias se llama eil22.txt
	el comando debe ser ./main eil22.txt y estar en el mismo directorio del archivo main.o, si no lo está se puede 
	ejecutar el cmoando como ./main "directorio". Se recomienda que el archivo esté en el mismo directorio.
3) Para ejecutar el algoritmo con heuristica se debe adicionar el sufijo -h, por ejemplo para la instancia eil22.txt 
	el comando sería ./main eil22.txt -h

4) Luego de correr el algoritmo, se genera un archivo de salida con el nombre de la instancia .out donde se pueden ver
	los resultados del algoritmo en el formato solicitado.
